# 🎭 RealReel: Real Eyes on Reel Lies

> ⚡ Lightweight deepfake detection system that works on CPU with >90% accuracy

RealReel is a lightweight deepfake detection system designed to identify manipulated images and videos efficiently on standard hardware. As deepfake technology becomes more advanced, detecting fake media is critical to prevent misinformation and fraud.

This project uses MobileNetV2 for feature extraction and a Linear Support Vector Machine (SVM) for classification, ensuring high accuracy with low computational cost.

## ⚡ Features
- Works on CPU (no GPU required)
- Supports image and video detection
- MobileNetV2 + SVM hybrid model
- Frame-based video analysis with majority voting
- Streamlit web interface

## 🎯 Applications
- Social media content verification
- Digital forensics
- Fake news detection
- Media authenticity verification
